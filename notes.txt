To connect react front end with express backend ->
1. download express generator in the main project application.
npx express-generator memories-backend

2. cd memories-backend
npm install
npm start

3.changed express port from 3000 to 9000 in bin > www

4. created new route in routes
added details in app.js

5. fetched api in react's app.js

6. installed and used cors in express
